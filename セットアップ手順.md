# LD Tool セットアップ手順

別PCでこのツールを使うための手順です。

---

## 事前インストール（1回だけ）

以下の3つを先にインストールしてください。

### 1. Lutron Designer
.pl ファイルの読み込みにSQL Serverが必要なため必須です。
インストールすると `MSSQL$LUTRON20XX` というSQL Serverサービスが自動的に入ります。

### 2. Python 3.x
- https://www.python.org/ からダウンロード
- インストール時に **「Add Python to PATH」にチェックを入れること**

### 3. ODBC Driver for SQL Server
- https://learn.microsoft.com/ja-jp/sql/connect/odbc/download-odbc-driver-for-sql-server からダウンロード
- `msodbcsql17.msi` または `msodbcsql18.msi` をインストール

---

## ツールのセットアップ

### 1. フォルダをコピー
このフォルダごと別PCに持っていきます（USBメモリ、ネットワーク共有など）。

### 2. セットアップ実行
PowerShellではなく **コマンドプロンプト（cmd）** で実行します。

PowerShellを開いている場合は、まず cmd に切り替えます：
```
cmd
```

次にフォルダに移動してセットアップを実行：
```
cd "コピーしたフォルダのパス"
setup.bat
```

**setup.bat が失敗する場合**は、以下のコマンドを手動で実行してください：
```
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
```

---

## 起動方法

コマンドプロンプトで：
```
cd "ツールのフォルダのパス"
start.bat
```

またはエクスプローラーから `start.bat` をダブルクリックしてもOKです。

ブラウザで `http://localhost:5000` にアクセスすると使えます。
